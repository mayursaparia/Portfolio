# 💼 Mayur Saparia — Portfolio Website

> Personal portfolio website built with HTML, CSS, and vanilla JavaScript. Deployed on Netlify.

🌐 **Live Site:** [mayurpersonalportfolio.netlify.app](https://mayurpersonalportfolio.netlify.app/)

---

## 👤 About

This is the source code for my personal portfolio website showcasing my experience, projects, and technical skills as a Full Stack Developer based in Toronto, ON.

---

## 📌 Sections

- **Hero** — Introduction with key impact metrics
- **Experience** — Career timeline (Northeastern, RBC, Cognizant, TechSecure)
- **Projects** — Selected technical projects
- **Skills** — Full technical stack with proficiency levels
- **Contact** — Email, LinkedIn, Phone, and Resume

---

## 🛠️ Built With

- HTML5 & CSS3 (no frameworks)
- Vanilla JavaScript
- [Google Fonts](https://fonts.google.com) — Syne + Inter
- Deployed on **Netlify** via GitHub integration

---

## 🚀 Run Locally
```bash
git clone https://github.com/mayursaparia/portfolio.git
cd portfolio
# Open index.html in your browser — no build step needed
```
